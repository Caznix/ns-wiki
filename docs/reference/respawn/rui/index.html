Rui

Functions for creating a rui, and methods of the rui object

// To create one, do:rui = RuiCreate( $"ui/assetname.rpak", topology, drawGroup, sortKey ) // sortkey = int to prevent z-fighting. higher -> in front
// You can then manipulate it using the following:RuiSetDrawGroup( rui, drawGroup )RuiSetString( rui, argName, value )RuiSetBool( rui, argName, value )RuiSetInt( rui, argName, value )RuiSetFloat( rui, argName, value )RuiSetFloat2( rui, argName, value )  // value is a vector; only x and y are usedRuiSetFloat3( rui, argName, value )RuiSetColorAlpha( rui, argName, color, alpha )  // color is a vector
// To destroy it, just do:RuiDestroy( rui )
